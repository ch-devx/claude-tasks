# Finalizar el sistema de Implantación

## Descripción
Para defender el proyecto de implantación de sistemas, debo verificar que el sistema funciona sin problemas. El dia 24 de junio es la defensa del proyecto. Si no lo hago el sistema podría tener fallas durante la defensa, aunque debo decir que el estado actual es robusto y las mejoras son mínimas, considero que cumple con lo esperado.
